# The Four Essential Freedoms of Free Software

A program is considered "free software" if the users possess the following four essential freedoms:

*   **Freedom 0:** The freedom to run the program as you wish, for any purpose.
*   **Freedom 1:** The freedom to study how the program works and modify it to suit your needs. Access to the source code is a prerequisite for this freedom.
*   **Freedom 2:** The freedom to redistribute copies of the program to help others.
*   **Freedom 3:** The freedom to distribute copies of your modified versions to others, allowing the entire community to benefit from your changes. Access to the source code is a prerequisite for this freedom.

This stands in contrast to proprietary software, which is protected by restrictive copyrights and typically hides its source code from users.
